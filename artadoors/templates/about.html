О сайте


Данный сайт является учебным проектом разработчика Скворцова Романа. Сайт на написан на Python с помощью Django, весь Frontend реализован в html шаблонах с помощью Bootstrap, CSS, AJAX (JavaScript).
Проект сделан монолитной архитектурой - всю логику обрабатывает Django, Nginx обрабатывает стили и изображения, Celery + Redis для уведомлений и электронной почты, база данных PostgreSQL. И все объединено в Docker-compose.
Проект относительно просто развернуть на любом сервере - подключаем сервер к github, настраиваем его, разворачиваем контейнеры Docker-compose, загружаем дамп базы данных, и все работает.

Разрабатываю проект, используя локальный сервер Django с настройками для разработки:
python manage.py runserver --settings=artadoors.settings.dev
Применяю систему контроля версий Git, можно вести разработку для тестирования новых функций и их последующего объединения (merge) в основную ветку. Настроен автоматический деплой: при пуше в master сервер обновляется, обеспечивая актуальность проекта без ручного вмешательства.
Код проекта открыт для просмотра на GitHub по ссылке https://github.com/roman-skwortsoff/artadoors.
